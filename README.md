# 📝 React 게시판 프로젝트 (React + TypeScript + Bootstrap)

## 💡 프로젝트 소개  
이 프로젝트는 **React + TypeScript + Bootstrap**을 사용한 간단한 **CRUD 게시판**입니다.  
나중에 **Spring Boot Backend**와 연결하여 API 연동도 진행할 예정입니다. 🚀  

## 🗝️ 주요 기능  
✅ 게시글 목록 조회  
✅ 새 게시글 추가  
✅ 게시글 삭제  
✅ Bootstrap을 활용한 UI 디자인
